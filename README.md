## 🔹 Overview  

**GreenNest** is a modern, elegant single-page application (SPA) designed for indoor plant enthusiasts who want to discover, care for, and purchase plants through a seamless digital experience. The platform combines an aesthetically pleasing, nature-inspired interface with practical functionality, enabling users to explore plant collections, learn essential care tips, and book expert consultations—all within a unified environment.  

The application is built with a strong focus on **user experience, performance, and clean frontend architecture**. Users can browse a curated list of indoor plants, view detailed information such as care level, availability, pricing, and ratings, and interact with the system through a smooth, uninterrupted SPA workflow. Each plant detail page is protected, ensuring that only authenticated users can access sensitive interactions like consultation booking.  

GreenNest implements a **secure and user-friendly authentication system** using Firebase, supporting both email/password and Google login. The system maintains persistent authentication, allowing users to navigate protected routes without disruption and automatically redirecting them to their intended pages after login.  

From a technical standpoint, the project showcases modern frontend development practices using **React and React Router**, enabling dynamic routing and component-based design. The UI is crafted using **Tailwind CSS and DaisyUI**, ensuring consistency, responsiveness, and a visually appealing layout across all devices. Animations and interactive elements powered by **Framer Motion and Swiper.js** further enhance the overall user experience.  

The platform dynamically loads plant data from structured JSON sources, demonstrating efficient data handling and smooth rendering without unnecessary reloads. Additional features like real-time toast notifications, protected routes, profile management, and form handling contribute to a complete and production-ready application.  

Overall, GreenNest represents a well-structured **frontend-focused SPA project** that highlights expertise in responsive design, authentication, stateful UI interactions, and user-centric application development—making it a strong example of modern web development aligned with industry best practices.
